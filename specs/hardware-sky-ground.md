# Железо: Небо + Земля + Ретранслятор

## Концепция: модульный конструктор

**Вставил чип — полетело.** SwarmHAL (Hardware Abstraction Layer) автоматически
обнаруживает любое железо и распределяет модели.

Поддерживаемые чипы: Nvidia (Jetson/RTX), Intel Arc, AMD ROCm, Rockchip NPU,
Hailo NPU, Apple MLX, чистый CPU.

Архитектура: `src/digital_twin/swarm_hal.py`

## Ключевой принцип: HAL

```
SwarmHAL.detect()  →  видит Nvidia? Intel? Rockchip? Hailo? CPU?
                        │
SwarmHAL.allocate() →  YOLO     → NPU (15ms)
                        Serafim  → CPU/NPU (50-400ms)
                        Thinker  → GPU/CPU (6s-45s)
                        CoderV2  → GPU/CPU (2s-40s)
                        │
                   Вставил чип → полетело.
```
