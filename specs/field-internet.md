# Полевой интернет: варианты для связи оператор ↔ рой

## Исходные данные

Наша система **НЕ гоняет видео** через интернет. Только:
- Тактические команды (сжатые через PromptCompressor: 233B → 44B, ×5.3)
- Телеметрия роя (wavelet DWT Хаара: 128 отсчётов × 4 канала → ~100B, ×8-12)
- Инструменты от Coder (код, разбитый на LoRa-пакеты по 51B)

**Требуемая полоса:** 1-5 Кбит/с вверх + 1-5 Кбит/с вниз.
Это НЕ видеопоток. Это текст и структурированные данные в wavelet-сжатии.

## Доступные варианты

### 1. Starlink (текущий стандарт)

| Параметр | Значение |
|----------|----------|
| Скорость | 50-200 Мбит/с ↓, 10-20 Мбит/с ↑ |
| Задержка | 25-50 мс |
| Терминал | 3.5 кг, 30×50 см |
| Цена | ~$600 терминал + $100-200/мес |
| Статус | **Отключают российским военным** (начало 2026) |
| Уязвимость | SpaceX контролирует доступ; терминалы пеленгуются |

**Вывод:** Ненадёжен для военных. Был основным, сейчас — резервный.

### 2. Бюро 1440 / «Рассвет» (российский LEO) 🥇

| Параметр | Значение |
|----------|----------|
| Спутников | 16 на орбите (март 2026), план 250+ к 2027 |
| Орбита | 800 км (низкая) |
| Задержка | ~50-70 мс |
| Частоты | Ka/Ku (устойчивее к РЭБ чем Starlink) |
| Стандарт | 5G NTN |
| Терминал | Разрабатывается (аналог Starlink-тарелки) |
| Статус | **Окна 15-20 мин над Украиной.** Коммерческий запуск 2027. |
| Партнёры | РЖД, МегаФон, Беларусь |

**Для нашей задачи:** Идеален в перспективе. 50-70ms задержки приемлемо.
Окна по 15-20 минут достаточно для пакетной отправки команд и получения телеметрии.
Рой автономен между окнами.

### 3. Sprint-030 (GEO, работает сейчас)

| Параметр | Значение |
|----------|----------|
| Спутники | Express-AM7, Express-AM44 (GEO, Airbus/Thales) |
| Скорость | 10 Мбит/с ↓, 1 Мбит/с ↑ |
| Задержка | 600+ мс (GEO — 36,000 км) |
| Антенна | 30 см (компактная, труднее запеленговать) |
| Производитель | ГК РЕЙС |
| Статус | **Развёрнут, работает.** Применяется с 2024. |
| Минус | ВСУ научились находить и уничтожать (414-я бригада) |

**Для нашей задачи:** 1 Мбит/с вверх с запасом покрывает наши 5 Кбит/с.
600ms задержка НЕ критична — оператор не пилотирует, а задаёт намерение.
Рой исполняет автономно, результаты приходят с задержкой.

### 4. МегаФон «Переносной комплекс» (через Бюро 1440) 🆕

| Параметр | Значение |
|----------|----------|
| Скорость | 150 Мбит/с ↓, 50 Мбит/с ↑ |
| Задержка | ~50-70 мс |
| Канал | Спутник Бюро 1440 → 4G/5G локальная сеть |
| Статус | **Представлен май 2026**, развёртывание 2026-2027 |
| Применение | Удалённые районы, частные сети |

**Для нашей задачи:** Перспективный. Локальная 4G-сота в поле + спутниковый uplink.
Запускающие с телефонами в локальной сети. Рой через LoRa подключён к базе.

### 5. Гонец-Д1М / Гонец-М1 (LEO, лёгкий)

| Параметр | Значение |
|----------|----------|
| Тип | Store-and-forward (пакетная передача) |
| Спутников | 12-24 (группировка «Гонец») |
| Скорость | 2.4-64 Кбит/с |
| Задержка | Минуты (не реальное время) |
| Статус | Работает. Военное применение подтверждено. |
| Терминал | Портативный, ~1-2 кг |

**Для нашей задачи:** Подходит для текстовых команд и сжатой телеметрии.
64 Кбит/с — с запасом для наших 5 Кбит/с.
Задержка в минуты приемлема для стратегических команд.

### 6. LTE/4G гражданский (запасной)

В прифронтовой полосе часто работает гражданский LTE.
- Резервный канал для полевой станции
- USB-модем + направленная антенна
- Может работать даже при слабом сигнале (наши данные крошечные)

## Влияние wavelet-сжатия

С нашим сжатием даже **Гонец (64 Кбит/с)** даёт:

```
64 Кбит/с ÷ 5 Кбит/с (наши потребности) = 12.8× запас

Один сеанс Гонца (1 минута):
  → 480 Кбайт данных
  → 10,900 тактических команд (по 44 байта)
  → 4,800 телеметрических пакетов (по 100 байт)

Достаточно для управления роем из 50+ дронов.
```

Без wavelet-сжатия (сырой JSON ~500 байт/команда):
```
Те же 480 Кбайт = всего 960 команд.
Разница: ×11 эффективности.
```

## Стратегия: двух- или трёхканальная связь

```
ОСНОвной:    Sprint-030 (GEO) — всегда работает, 600ms
РЕЗЕРВ:      Гонец — если Sprint заглушен/уничтожен
БУДУЩЕЕ:     Бюро 1440 — как спутников станет >100
АВАРИЯ:      Гражданский LTE — USB-модем
```

**Ключевой принцип:** рой автономен и без связи.
Интернет нужен только для ОПЕРАТОРА (дома) ↔ ПОЛЕВОЙ СТАНЦИИ.
Если связи нет — полевая станция + Serafim продолжают миссию.
